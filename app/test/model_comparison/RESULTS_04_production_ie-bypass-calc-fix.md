## 3. T1 답변 분석 — 채점기준 재검토

### 3-1. 5회 답변 공통 패턴

5회 모두 동일한 구조로 답변:
- **Case 1 (본인)**: A가 재고 통제 보유 → 120원 총액 인식, 20원 판매비
- **Case 2 (대리인)**: B가 독립 유통 → A는 100원(수수료) 인식
- **판단 기준 안내**: 통제 여부 (가격결정권, 재고위험, 반품위험)
- **세금계산서 지적**: 법적 명의 이전 ≠ 수익 인식 판단

### 3-2. 기존 채점기준의 문제

기존 채점기준 3번: "이후 결론을 본인인 경우 120원으로 총액제시 했는지"
→ **120원 단정을 요구**하지만, 질문에 위탁판매 확정 근거가 없음.

질문 원문 분석:
- "A가 B에게 재화를 100원에 공급" — 단순 공급, 위탁 약정 명시 없음
- "B는 최종 고객 C에게 120원에 판매" — B의 독립 판매일 수도, 위탁일 수도 있음
- 세금계산서 발행 → 법적 명의 이전은 판단 기준 아님

**결론: 챗봇의 양쪽 케이스 제시가 정답.** 모범답안이 과도하게 단정적.

### 3-3. 수정 채점기준

```
수정 전: "이후 결론을 본인인 경우 120원으로 총액제시 했는지"
수정 후: "A가 본인인 경우 120원 총액, 대리인인 경우 수수료 인식이라는 양쪽 케이스를 모두 제시했는지"
```

→ Step 2에서 `model_comparison_question.py` 반영
